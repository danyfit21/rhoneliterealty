# Diego Rhon Website

Sitio estatico listo para publicar en GitHub Pages.

## Publicar en GitHub

1. Sube todo el contenido de esta carpeta al repositorio.
2. En GitHub, ve a Settings > Pages.
3. En Branch, selecciona `main` y carpeta `/root`.
4. Guarda los cambios.

El archivo principal es `index.html`.

## Estructura

- `index.html`: sitio completo.
- `assets/`: imagenes locales usadas por el sitio.
- `.nojekyll`: evita que GitHub Pages procese la carpeta con Jekyll.
